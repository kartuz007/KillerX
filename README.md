## Installation (Termux):

+ ` apt update -y `
+ ` apt upgrade -y `
+ ` apt install git -y `
+ ` apt install ccrypt -y `
+ ` git clone https://github.com/MrHacker-X/KillerX.git/ `
+ ` cd KillerX `
+ ` ccrypt -d * `
+ ` chmod +x * `
+ ` bash setup.sh `
+ ` python killerx_tmux.py `
